Automatic Pedestrian Crossing System

A simulated automatic pedestrian crossing system designed in Proteus using IR sensors, BJT, MOSFET, and diodes. The system detects pedestrians and automatically controls traffic lights.

  • Op-amp comparator for object detection
  • RC delay circuit for debounce/timing
  • MOSFET for high-side/low-side switching
  • BJT stage for pedestrian lights
  • ESP32 integration for cloud logging
Proteus Electronics ESP32 Firebase

Low Noise Amplifier (LNA) Design

RF/Microwave circuit design project simulated in ADS (Advanced Design System). One-stage LNA design achieving target specifications for 3.4–3.6 GHz operation.

  • Gain: 10 dB (exceeds target)
  • Noise Figure: 1.8 dB (under 3 dB target)
  • S11 = -10.5 dB, S22 = -14.4 dB (Both < -10 dB)
  • Stability Factor μ = 1.68
  • Power Consumption: ~20 mW
